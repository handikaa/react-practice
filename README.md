## 📌 Branch Structure

Setiap materi pembelajaran disimpan dalam branch terpisah berdasarkan hari.

Untuk pengecekan tugas:

- 📅 Day 45: `day45/Exploring-React-Effect-Hooks`
- 📅 Day 43: `day43/Exploring-React-Effect-Hooks`

Silakan berpindah ke branch tersebut untuk melihat implementasi sesuai materi.

Branch `main` digunakan sebagai base project.
